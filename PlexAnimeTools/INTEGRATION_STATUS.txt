QUICK START GUIDE
=================

   ------------
   Import-Module .\PlexAnimeTools.psd1

   -------------------
   Edit Config/default.json

   -----------------
   .\Test-Integration.ps1

   ---------------------------

   ---------------
   Add library in Plex pointing to output folder
   Scan library
   Enjoy!
